Hugo Install

Download

MAC OS 기준

brew install hugo

blog 뼈대 만들기

hugo new site BLOG

~/BLOG 디렉토리 밑에 hugo blog 의 빼대가 생김

tree -d BLOG

BLOG
├── archetypes
├── content
├── data
├── layouts
├── static
└── themes

config.toml config 파일

git init

Themes 적용

https://themes.gohugo.io/

사이트에 들어가 테마를 고르고

git submodule add https://github.com/lxndrblz/anatole.git themes/anatole

./config.toml 파일 수정 ( 테마에서 제공하는 sample config 를 가지고 수정 : 대부분 의 경우 clone 받은 테마 디렉토리 안에 있다. )

baseurl 부분 삭제
baseURL = ""

cp -rfp themes/anatole/exampleSite/config.toml . cp -rfp themes/anatole/exampleSite/static .

Local 에서 확인

hugo server -D

-D 는Drafts 파일 까지 다 보여줌. Drafts 파일이란 작업중인 글을 말한다. 새로운 글을 작성시 문서 첫부분에 Drafts 설정 부분이 있다.

github 에 올려거 인터넷으로 확인 하기

Before proceeding run rm -rf public to completely remove the public directory git submodule add https://github.com/taein2301/taein2301.github.io.git public git remote add origin https://github.com/taein2301/blog.git

hugo 명령어 수행 > public 폴터가 생기고 해당 폴더를 github 에 올려서 인터넷으로 확인. 
